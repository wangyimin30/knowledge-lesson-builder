💡 本次学习总览

辩论是以驳倒对方观点、确立己方观点为目的的口语交际活动，是初中阶段重要的口语实践形式。本学案共5个模块，依次训练：辩题判断、立论构建、反驳应对、总结陈词，最后综合实战演练。学完后，学生将能够独立分析辩题、撰写辩论稿、有理有据地开展辩论。

模块一 辩题判断

学习目标：能识别辩题是否具备争议性、可辩性和现实意义，并能根据反方辩题补写正方辩题

📚 知识要点

辩论的起点是一个好辩题。好辩题必须满足三条标准：贴近学生生活、有可辩性（双方各有充分论据，不存在明显正误）、有现实意义和启发性。事实性问题（有标准答案）不适合辩论。

  • 三条标准：贴近生活 + 有可辩性 + 有现实意义
  • 分析辩题4步法：①圈关键词 → ②做有利于己方的定义 → ③确定比较标准 → ④预判对方反驳
  • 补写正方辩题：观点与反方相反，表述正面，结构如"……是……，值得……"或"……有利于……"

典型易错点：A"阅读经典很有意义"几乎没有争议，不适合辩论；事实性判断如"叙事者'我'是不是作者"有标准答案，同样不适合。

✅ 模块一学习目标达成自评

模块二 立论构建

学习目标：能运用"四段式"结构，明确立场，搭建论点+论据+论证框架，写出完整立论稿

📚 知识要点

立论是辩论的核心，一辩需要开门见山亮出观点，并用分论点+论据加以支撑。论据必须直接指向论点，不能跑题。

  • 四段式结构：【开场】问候+亮观点 → 【分论点1】+事例/数据 → 【分论点2】+事例/数据 → 【结尾】总结重申
  • 辩手分工：一辩立论；二辩/三辩补充并主攻；四辩总结陈词
  • 辩论五原则：表达清晰·声情并茂 / 观点鲜明·协同作战 / 善于聆听·快速反应 / 语言严谨·简洁有力 / 仪态风度·胜不骄败不馁

📝 真题训练

2024年杨浦一模 · 第22题
辩题：顺境更有利于成长成才（正方）vs 逆境更有利于成长成才（反方）

正方一辩论述：孙悟空顺利练就"火眼金睛"，习得"七十二变"，顺利取得"金箍棒"……顺境促进孙悟空提升实力，足以证明顺境更有利于成长成才。

任务：你是反方一辩，请写一段立论稿（100字左右），反驳对方例子并正面论证逆境更有利于成长。
【参考答案】 我方认为，逆境更有利于成长。对方说孙悟空顺利练就本领，但火眼金睛恰恰是在太上老君的炼丹炉中炼成的，那是九死一生的逆境！他被压五行山下五百年，才磨去野性，最终成佛。可见，真正的蜕变来自逆境的锤炼，而非顺境的安逸。（约90字）

评分说明：反驳对方例子（3分）；正面立论，论点清晰（3分）；语言流畅有力（2分）；附：可引用"生于忧患，死于安乐"等名言加分。
空中课堂·"以成败论英雄"练习
正方论点：①以成败论英雄将追求成功转化为精神追求；②具有广泛现实意义，鼓励全社会积极进取。

任务：你是反方一辩，请写一段立论稿（150字以内），反驳正方并阐明己方观点"以成败论英雄不可取"。
【参考答案】 我方认为，以成败论英雄是不可取的。第一，成败受太多客观因素影响——周瑜以火烧赤壁力挫曹军，是何等英雄，却含恨"既生瑜何生亮"，我们能否认他是英雄吗？第二，以成败论英雄会诱导人们不择手段，只看结果不问过程。英雄的本质是超越成败的精神，而不是冰冷的数字。因此，以成败论英雄不可取。

评分说明：观点鲜明（2分）；两个分论点各有论据（各3分）；总结有力（2分）。
迁移任务
辩题：自媒体时代更容易接近真相（正方）

任务：写一段正方一辩立论稿（100-150字）。
【参考答案（示例）】 我方认为，自媒体时代更容易接近真相。第一，自媒体打破了传统媒体的信息垄断，多元视角让真相无处遁形——许多社会事件首先由普通网友曝光，再经专业媒体核实。第二，信息传播速度极快，谣言往往迅速被辟谣，真相在博弈中凸显。因此，自媒体为普通人提供了接近真相的便捷通道。

评分说明：开放题，言之有理即可给分。关注点：是否有明确立场、论据是否能证明论点、逻辑是否自洽。

✅ 模块二学习目标达成自评

模块三 反驳应对

学习目标：能快速识别辩论准备与非准备环节，判断论据有效性，为辩题找到有力的反驳论据

📚 知识要点

反驳是辩论的核心技能，需要快速抓住对方的逻辑漏洞或论据缺陷，从驳论点、驳论据、驳论证三个方向展开。

  • 反驳三方向：驳论点（对方观点本身有误）/ 驳论据（对方举例不能证明观点）/ 驳论证（对方推理过程有漏洞）
  • 好论据的标准：必须直接支持己方观点，类型包括：名家评价、相关事实案例、制度规则分析
  • 辩论准备环节：属于准备的有——组织队伍、分析辩题、整理材料、设计提纲、对辩练习；不属于准备的有——调查大众观点、与辩论无关的事务性工作

✅ 模块三学习目标达成自评

模块四 总结陈词

学习目标：能运用"四步法"撰写总结陈词，做到回应全场、升华价值观、留下深刻印象

📚 知识要点

总结陈词是辩论的最后一击，由四辩完成。好的总结陈词不仅要总结己方观点，更要有效回应对方，并将辩题升华到更高的社会/人文价值层面。

  • 四步法：①提炼核心争议 → ②抓住对方逻辑漏洞 → ③总结本方核心论点 → ④升华价值（社会/人生/时代）
  • 好陈词的特征：观点鲜明、语言有力（善用反问/排比）、升华自然、情感有感染力
  • 评价维度：提炼与概括 / 反驳与针对性 / 逻辑与升华 / 语言与表达
